Branch Daily Checklist

1) Branch opened on time? (क्या ऑफिस समय पर खुला?)
2) Water Dispenser working? (क्या वाटर डिस्पेंसर काम कर रहा है?)
3) Tea Coffee Machine working? (क्या चाय कॉफी मशीन काम कर रही है?)
4) HK person / Security Guard reported on time?
5) Biometric machine working?
6) Biometric door lock working?
7) CCTV System is working?
8) All AC is working?
9) UPS is working?
10) Any incident happened in branch? (क्या शाखा में कोई घटना हुई?)
11) Washroom Checklist updated?
12) Electricity bill submitted to admin?
13) Leakage / Seepage / bad smell? (क्या लीकेज / दुर्गंध की समस्या है?)
